Этот визуализатор имитирует выполнение некоторого подмножества языка Python 3.2. Реальное выполнение кода происходит на Python 2.7, поэтому можно наблюдать некоторые ошибки, связанных с переводом кода между версиями 2 и 3 до и после выполнения программы. Для запуска реальных приложений используйте Python 3, в качестве среды разработки рекомендуем Wind IDE 101.
