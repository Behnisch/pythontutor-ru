Name:
Округление по российским правилам

Statement:
По российский правилам числа округляются до ближайшего целого числа,
а если дробная часть числа равна 0.5, то число округляется вверх.

<p>Дано неотрицательное число <code>x</code>, округлите его по этим правилам.
Обратите внимание, что функция <code>round</code> не годится для этой задачи!

Test:
2.3

Answer:
2

3

3

3
