Вы открыли первый урок нашего учебника языка Python. Python — это современный язык, на котором просто и приятно писать программы. Подробнее о нём вы можете прочитать на Википедии. Мы будем изучать версию языка Python 3.

Мы будем писать программы, которые принимают какие-либо данные на вход, каким-то образом их перерабатывают и выводят что-то. При запуске на компьютере такие программы считывают данные, поданные пользователем с клавиатуры, а результат выводят на экран.

Для вывода в Python есть функция print(). Внутри круглых скобок через запятую можно написать то, что мы хотим вывести. Вот пример программы, из которого видно, что Python можно использовать как простой калькулятор. {% program %} print(3 * 7) print(17 - 9) print(2 ** 100) # это возведение в степень print(37 / 3) # это "обычное" деление print(37 // 3) # это нахождение частного от деления # аналогично операции div в Паскале print(37 % 3) # это нахождение остатка от деления # аналогично операции mod в Паскале {% endprogram %}

Для ввода данных можно использовать функцию input(). Эта функция считывает из ввода одну строку и возвращает её.

Напишем программу, которая считывает имя пользователя и приветствует его: {% program %} print('Как вас зовут?') name = input() print('Здравствуйте, ' + name + '!') {% inputdata %} Аристарх {% endinputdata %} {% endprogram %}

Попробуем написать программу, которая считывает два числа и выводит их сумму. Для этого считаем два числа и сохраним их в переменные a и b, пользуясь оператором присваивания =. Оператор присваивания в Python связывает имя, стоящее слева от него, с объектом, который получается в результате вычисления того, что стоит справа от него. Запустите следующий пример в визуализаторе и посмотрите на результаты его работы. {% program %} a = input() b = input() s = a + b print(s) {% inputdata %} 5 7 {% endinputdata %} {% endprogram %}

Как видите, Python вывел 57 вместо желаемого 12. Это произошло потому, что Python в третьей строчке «сложил» две строки, а не два числа. В Python сложение строк — это приписывание второй строки к первой. Обратите внимание, что в визуализаторе содержимое переменных a и b заключено в кавычки.

Python хранит данные в объектах, и разные объекты могут иметь разные типы. Строки хранятся в объектах типа str, целые числа хранятся в объектах типа int, дробные числа (так называемые вещественные числа) — в типе float. Чтобы преобразовать строку из цифр в число, нужно воспользоваться функцией int(). Например, int('23') вернет число 23. Заметьте, что функция перевода в тип int называется так же, как и сам тип (это неспроста).

Вот пример правильной программы, которая считывает два числа и выводит их сумму: {% program %} a = int(input()) b = int(input()) s = a + b print(s) {% inputdata %} 5 7 {% endinputdata %} {% endprogram %}
