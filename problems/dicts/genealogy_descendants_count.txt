Name:
Родословная: число потомков

Statement:
<p>Для каждого элемента дерева определите число всех его потомков (не считая его самого).</p>

<p>
Формат выходных данных совпадает с
<a href="http://pythontutor.ru/lessons/sets/problems/genealogy_level_count/">задачей</a>. 
</p><p>
Выведите список всех элементов в лексикографическом
порядке, для каждого элемента выводите количество всех его потомков.
</p>

Test:
9
Alexei Peter_I
Anna Peter_I
Elizabeth Peter_I
Peter_II Alexei
Peter_III Anna
Paul_I Peter_III
Alexander_I Paul_I
Nicholaus_I Paul_I

Answer:
Alexander_I 0
Alexei 1
Anna 4
Elizabeth 0
Nicholaus_I 0
Paul_I 2
Peter_I 8
Peter_II 0
Peter_III 3
